# AnimalPedia

AnimalPedia is an Animal Encyclopedia application designed specifically for children to learn and explore various animals in the world interactively and enjoyably.

## Installation

1. Download the APK file from [here](https://github.com/rahmabintangpratama/AnimalPedia/releases/download/v1.0.0/AnimalPedia.apk).
2. Install the APK file on your Android device.
3. Open AnimalPedia and start exploring the world of animals!

Thank you for using AnimalPedia! We hope this application helps children learn and appreciate the biodiversity in our world.
